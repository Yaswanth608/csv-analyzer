# AI ZIP Folder CSV Comparison Project

This project compares two ZIP folders containing CSV files. It allows you to input two ZIP files and a sender address, then extracts all CSVs, searches for the sender address in all CSVs from both ZIPs, and outputs the total count of occurrences.

## Features
- Accepts two ZIP files as input
- Extracts and processes all CSV files inside each ZIP
- Searches for a given sender address in all CSVs
- Outputs the total count of occurrences of the sender address

## Usage
1. Place your two ZIP files in the project directory.
2. Run the main script and provide the ZIP file paths and sender address when prompted.
3. View the output count.
